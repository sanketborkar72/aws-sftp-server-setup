# AWS SFTP Server (Ubuntu 22.04)

This project contains the manual setup guide for deploying a secure SFTP server on AWS EC2.

## Connection Details
- **Host:** AWS Elastic IP
- **Port:** 22
- **Username:** SFTP
- **Password:** xyz@123

## Features
- Chroot Jail for security
- Static Elastic IP
- Pre-configured for WinSCP / FileZilla for access it from windows system.
